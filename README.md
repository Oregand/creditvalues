# creditvalues.net

An archived, styled reproduction of GitLab's **CREDIT** values
(Collaboration · Results · Efficiency · Diversity, Inclusion & Belonging ·
Iteration · Transparency), served at [creditvalues.net](https://creditvalues.net).

The content is reproduced from the [GitLab Handbook](https://gitlab.com/gitlab-com/content-sites/handbook/-/blob/bc2282a09667ea2358091065f0dd4431d16d4040/content/handbook/values/_index.md)
and is © GitLab B.V., licensed under [CC BY-SA 4.0](https://creativecommons.org/licenses/by-sa/4.0/).
This site is not affiliated with or endorsed by GitLab.

## Local development

A plain Jekyll site with hand-written CSS (`css/handbook.css`) that recreates
the clean, typographic look of the original GitLab handbook (Inter font,
orange blockquote accents, GitLab-blue links). No build step required.

```sh
bundle exec jekyll serve
```

Deployed via GitHub Pages from the `main` branch.
